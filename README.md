## 如何在Latex中使用Emoji
1. 使用LuaLatex编译器
2. 在Latex文件中引入库
- 导入emoji包：
    > \usingpackage{emoji}
- 可选的emoji家族
    > \setemojifont{TwemojiMozilla}

参考：[Overleaf guilds](https://www.overleaf.com/learn/latex/Questions/Inserting_emojis_in_LaTeX_documents_on_Overleaf)
